# App Project

## Installation

### 1. Install uv Package Manager

Install uv using the official installer:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Or using pip:

```bash
pip install uv
```

### 2. Clone the Project

```bash
git clone <repository-url>
cd app-project
```

### 3. Sync Dependencies

Sync project dependencies using uv:

```bash
uv sync
```

### 4. Run the Project

Execute the main application:

```bash
uv run main.py
```
